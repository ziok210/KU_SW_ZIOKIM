# Life Schedule

To-Do, 메모, 일기 기능을 하나로 통합한 개인 일정 관리 프로그램

## 프로젝트 소개

Life Schedule은 사용자의 일정을 효율적으로 관리하기 위한 데스크톱 애플리케이션입니다

할 일(To-Do), 메모(Memo), 일기(Diary)를 하나의 프로그램에서 관리할 수 있으며, 또한 검색 기능과 다크 모드를 지원하여 편리한 사용자 경험을 제공합니다

### 주요 기능

* To-Do 관리

  * 할 일 추가, 수정, 삭제
  * 완료 여부 체크
  * 마감일 및 우선순위 설정

* 메모 관리

  * 메모 작성, 수정, 삭제
  * 제목 및 내용 저장

* 일기 관리

  * 날짜별 일기 작성
  * 달력 기반 조회

* 검색 기능

  * To-Do, 메모, 일기 통합 검색

* 설정 기능

  * 라이트 모드 / 다크 모드 지원

* 데이터 저장

  * JSON 파일 기반 자동 저장

---

## 사용 방법

### To-Do 사용

1. To-Do 메뉴 선택
2. 제목, 마감일, 우선순위 입력
3. 추가 버튼 클릭
4. 완료 체크 또는 삭제 가능

### 메모 사용

1. 메모 메뉴 선택
2. 제목과 내용 입력
3. 저장 버튼 클릭

### 일기 사용

1. 일기 메뉴 선택
2. 달력에서 날짜 선택
3. 일기 작성 후 저장

### 검색 사용

1. 검색 메뉴 선택
2. 키워드 입력
3. 관련 To-Do, 메모, 일기 검색 결과 확인

---

## 입력 / 출력

| 기능    | 입력            | 출력        |
| ----- | ------------- | --------- |
| To-Do | 제목, 마감일, 우선순위 | 할 일 목록    |
| 메모    | 제목, 내용        | 저장된 메모    |
| 일기    | 날짜, 내용        | 날짜별 일기    |
| 검색    | 키워드           | 관련 데이터 목록 |
| 설정    | 테마 선택         | 다크/라이트 모드 |
